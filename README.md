# star-wars-crud

> CRUD of Star Wars characters with a clean user interface written in Typescript.

Features:

- Intuitive UX/UI
- Performant
- Hexagonal Arquitecture
- React hooks
- UI components
- Custom Properties CSS
- Mocked API
- Design system
- Responsive
- Clean folder structure

## Getting Started

### Installation

```sh
npm i
```

### Run mocked api server

```sh
npm run api-start
```

### Run web server

```sh
npm run start
```

## Development

### Run lint

```sh
npm run lint
```

### Run format

```sh
npm run format
```

### Run tests

```sh
npm test
```

### Build

```sh
npm run build
```
